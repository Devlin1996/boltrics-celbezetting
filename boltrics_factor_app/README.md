# Boltrics Celbezetting App

Start lokaal:

```cmd
py -m streamlit run app.py
```

Upload je Boltrics Handling Unit export. Kolommen worden gelezen als:
- Kolom A: HU barcode
- Kolom B: Location No.
- Kolom C: Handling Unit Type Code

In de zijbalk kun je palletfactoren en celinstellingen aanpassen.
